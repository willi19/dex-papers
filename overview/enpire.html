ENPIRE: Agentic Robot Policy Self-Improvement in the Real World

2026NVIDIA / CMU / UC Berkeleyself-improvingagenticRLdexterous

ENPIRE is a harness that lets frontier coding agents (e.g. Codex/GPT-5.5, Claude Code, Kimi Code) autonomously improve robot manipulation policies on real hardware through a closed feedback loop. It turns real-world robot learning into a controllable optimization procedure the agent can manage, moving the self-improvement loop off the simulator and onto physical robots. The agent designs its own tests rather than just its own rewards: it resets the environment, searches the literature, implements and builds infrastructure, trains and deploys policies, self-verifies, analyzes logs, rewrites code, and repeats until the policy is reliable. The work is a collaboration between the NVIDIA GEAR Lab, Carnegie Mellon University, and UC Berkeley, with equal advisors Jim Fan, Yuke Zhu, and Guanya Shi.

Four core modules spell ENPIRE: EN (Environment) for automatic reset and verification interfaces; PI (Policy Improvement) that refines policies from failure analysis and logs; R (Rollout) that executes trials on single or parallel robots; and E (Evolution) where agents analyze data, consult literature, and improve algorithms. The self-improvement loop runs on real hardware instead of in simulation, with the agent designing its own verification tests rather than only its own reward functions. pass@8 metric measures emergent retry and recovery within single rollouts, allowing up to 8 in-context retries per subtask. Coding agents tested include Codex (GPT-5.5), Claude Code, and Kimi Code variants, run across fleet sizes of 1, 4, and 8 robots operating in parallel. The framework minimizes human effort by handing the entire research-and-iteration loop to the agent (reset, search, implement, train, deploy, verify, analyze, rewrite).

Achieves a 99% pass@8 success rate across challenging real-world manipulation tasks. Demonstrates autonomous algorithmic improvements with measured gains: Online RL (+3.8 pp), BC regularization (+10.8 pp), and controller compensation (+1.3 pp). Validates the system on high-precision real-world tasks including GPU insertion, fine pin organization, and zip-tie cutting performed by the robot on its own. Shows scaling across fleets of 1, 4, and 8 physical robots running rollouts in parallel. Shows the approach generalizes across multiple frontier coding agents (Codex/GPT-5.5, Claude Code, Kimi Code).

Demo videos show bimanual real-robot tasks: Push-T, pin insertion, GPU insertion, and zip-tie tying and cutting, plus RoboCasa simulation tasks (coffee setup, mug placement, cabinet/drawer opening, and mixer/stove/sink operations). The agent solves high-precision tasks like installing GPUs and tying/cutting zip-ties entirely on its own.
