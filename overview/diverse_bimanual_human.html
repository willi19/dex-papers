Learning Diverse Bimanual Dexterous Manipulation Skills from Human Demonstrations (BiDexHD)

2024Peking Universityimitationhuman-demosbimanual

BiDexHD ("Learning Diverse Bimanual Dexterous Manipulation Skills from Human Demonstrations", arXiv 2410.02477) is a framework for efficiently learning many bimanual dexterous manipulation skills from abundant human demonstrations rather than from per-task hand-engineered rewards. It addresses the high-dimensional action space and task complexity of two-hand dexterous control, and the limited task diversity of existing benchmarks, by automatically constructing tasks from existing bimanual human-object datasets and solving them with a single unified pipeline. It couples teacher-student policy learning with a general two-stage reward function shared across tasks, so that scalable multi-task learning becomes feasible. The work targets progress toward universal/general-purpose bimanual dexterous manipulation, demonstrating both strong training-task performance and competitive zero-shot generalization to unseen tasks.

Unifies automatic task construction from existing bimanual human-object datasets (TACO) so numerous tasks can be set up without per-task manual design. Uses a general two-stage reward function shared across tasks with common behaviors: a first alignment stage (approaching, lifting, and bonus rewards to align the simulation with the demonstration) and a second tracking stage (exponential reward for precise trajectory following), avoiding intricately tuned per-task rewards. Teacher-student learning: teachers learn state-based multi-task policies (trained with Independent PPO / IPPO), and the student distills them via DAgger into a vision-based policy conditioned on point clouds and future object positions. Embodiment is two dexterous hands (LEAP Hand) on two robotic arms, simulated in IsaacGym, learning to use tools to manipulate target objects. Tasks are auto-constructed from human demonstrations and grouped by shared behavior so a single reward/pipeline scales across the whole task set. Designed for zero-shot generalization: policies are evaluated on held-out unseen tasks, not just the trained ones.

Introduces BiDexHD, a unified framework that converts human-demonstration datasets into trainable tasks and solves all of them with shared teacher-student learning and a single two-stage reward, removing per-task manual reward engineering. Evaluates on the TACO dataset spanning 141 auto-constructed tasks across 6 categories, using an 80%/20% train/test split. Achieves a 74.59% task fulfillment (tracking) rate on trained tasks. Achieves a 51.07% task fulfillment rate on unseen tasks, demonstrating competitive zero-shot generalization. Demonstrates scalable multi-task RL for bimanual dexterous skills without narrowly tailored rewards, positioning the work as a step toward universal bimanual dexterous manipulation. Distills state-based multi-task policies into a deployable vision-based policy (point clouds + future object positions) via DAgger.

It learns 141 bimanual dexterous tool-use tasks (across 6 TACO categories such as pouring, emptying/dusting, and tool-strike/screwing manipulations) auto-constructed from human demonstrations, using two LEAP-hand-equipped arms in IsaacGym. The project page (https://sites.google.com/view/bidexhd) shows video demos of these TACO tasks (e.g., pouring with a cup/teapot, emptying a bowl, dusting with a brush, hammering, screwdriving) as well as additional ARCTIC-dataset demos (mixer, capsule machine, box flipping, ketchup lifting).
