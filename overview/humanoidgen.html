HumanoidGen: Data Generation for Bimanual Dexterous Manipulation via LLM Reasoning

NeurIPS 2025TeleAI (China Telecom) / Fudan Universitydata-generationLLM-reasoningdiffusionbimanual

HumanoidGen is an automated simulation framework for scene generation, demonstration collection, and data generalization for bimanual humanoid robots equipped with dual arms and dexterous hands. It composes atomic dexterous operations and uses an LLM to reason over spatial annotations of assets and hands, generating relational constraints on arm movements to plan tasks while avoiding collisions. For long-horizon tasks with insufficient annotation, it augments LLM reasoning with a Monte Carlo Tree Search variant via a Segment-Truncate-Combine-Resume mechanism. The output is high-quality demonstration data across diverse scenarios that can be used to train manipulation policies.

Spatial annotation of both assets and dexterous hands provides the grounding an LLM needs to plan dexterous bimanual operations. Tasks are built from reusable atomic dexterous operations, with the LLM generating relational action constraints between the two arms plus collision avoidance. Automated task decomposition turns high-level goals into constraint-based motion plans, enabling autonomous demonstration collection in simulation. A Monte Carlo Tree Search variant (Segment-Truncate-Combine-Resume) enhances LLM reasoning for long-horizon tasks where single-pass planning fails. Scenes are scaled (room-level scaling drawing on RoboCasa assets) to diversify scenarios and test generalization. The generated dataset is used to train 2D and 3D diffusion policies, whose performance scales with demonstration count.

HGen-Bench benchmark of 20 diverse bimanual dexterous manipulation tasks spanning single-arm, bimanual, long-horizon, and articulated-object challenges. Spatial annotation + LLM constraint-generation framework producing relational action constraints with collision avoidance for dual-arm dexterous planning. MCTS-enhanced LLM reasoning (Segment-Truncate-Combine-Resume); e.g., Blocks Stack Hard rises from 18.3% to 98.3% success with MCTS (N=12), Pyramid Stack from 13.3% to 90.0%. Demonstration data of 100 samples per task (about 2,000 demonstrations across the benchmark) plus over 120 diverse scaled scenes. Evidence that diffusion policies scale with generated data: DP3 on Open Box Hard improves 95.6% (20 demos) to 98.3% (100 demos); 2D DP on Open Box Hard rises 11.1% to 100%. Public release of dataset and models on Hugging Face and code on GitHub (TeleHuman/HumanoidGen).

Automatically creates bimanual dexterous manipulation tasks and collects scalable, high-quality demonstration data in simulation, which is then used to train 2D and 3D diffusion policies that improve as the generated dataset grows.
