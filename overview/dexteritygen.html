DexterityGen: Foundation Controller for Unprecedented Dexterity

RSS 2025UC Berkeley (BAIR) and Meta FAIRRLteleoperationdexterousgenerative

DexterityGen (DexGen) is a foundation controller for robotic dexterous manipulation that combines reinforcement learning with human teleoperation. It uses RL to pretrain a large-scale, multi-task dataset of dexterous motion primitives such as in-hand rotation and translation. A generative model is then trained on this dataset to translate coarse, potentially unsafe motion commands from an external source (a human teleoperator or another policy) into safe, fine-grained actions. This separation lets RL handle low-level motion primitives while humans provide high-level coarse commands for complex, long-horizon tasks.

Hybrid RL + teleoperation design: RL learns low-level dexterous motion primitives, while humans supply coarse high-level motion commands for long-horizon tasks. RL pretraining of large-scale multi-task dexterous primitives (e.g., in-hand rotation and translation) builds a reusable motion dataset. A generative model learned from that dataset acts as the foundation controller, mapping unsafe/coarse commands to safe, fine actions. Functions as a general-purpose controller that realizes input dexterous manipulation commands across diverse tasks. Greatly improves manipulation stability, reducing dropped objects during teleoperated dexterous control. Enables grasp transitions (e.g., pinch to power grasp) and complex tool use as composable behaviors.

Introduces DexGen, a foundation controller that translates coarse, unsafe motion commands into safe, fine-grained dexterous actions. Improves manipulation stability by 10-100x, measured as the duration of holding objects, across diverse tasks. Demonstrates unprecedented dexterous tool use for the first time, including pen, syringe, and screwdriver. Shows diverse object reorientation across objects of varying sizes and properties. Demonstrates assembly tasks and grasp transitions (pinch to power grasp) via the unified controller. Pretrains the controller on a large-scale multi-task dexterous manipulation dataset of RL-learned motion primitives.

It enables a multi-fingered robot hand to perform stable, high-precision dexterous manipulation -- object reorientation, assembly, and tool use with a pen, syringe, and screwdriver -- by correcting coarse human teleoperation or policy commands into safe fine motions.
