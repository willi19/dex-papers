UniDexGrasp++: Improving Dexterous Grasping via Geometry-aware Curriculum and Generalist-Specialist Learning

ICCV 2023PKU / academicRLgraspinggeneralist-specialist

UniDexGrasp++ (ICCV 2023) is an object-agnostic pipeline for learning a single universal dexterous grasping policy from realistic point-cloud observations plus proprioception in a table-top setting. It trains and evaluates on 3000+ different object instances presented in random poses, following the convention of first learning a state-based teacher policy and then distilling it into a vision-based student policy. Its core idea is to use object geometry to structure the learning process, via two proposed methods: Geometry-aware Curriculum Learning (GeoCurriculum) and Geometry-aware iterative Generalist-Specialist Learning (GiGSL). The work was a finalist for the ICCV 2023 Best Paper award.

Pretrain a point-cloud autoencoder and use its bottleneck feature as a metric of the task space, so that geometrically similar grasping tasks are clustered together. GeoCurriculum (Geometry-aware Curriculum Learning): progressively expand training from single/few objects to diverse instances, ordering the curriculum by geometry rather than treating all objects uniformly. GiGSL (Geometry-aware iterative Generalist-Specialist Learning): alternately train specialist policies on geometry-defined task subspaces and distill them back into an improved generalist policy, iterating to boost cross-object generalization. Two-stage learning: a state-based RL policy is learned first, then distilled into a vision-based policy that operates from point clouds + proprioception (no privileged state at test time). Geometry features guide both how the task space is partitioned and how learning progresses, which is the main lever for generalizing to unseen objects and poses.

A universal dexterous grasping policy that achieves 85.4% success on the train set and 78.2% success on the test set across 3000+ object instances with random poses. Improves over the prior UniDexGrasp baseline by +11.7% (train) and +11.3% (test) in success rate. Introduces GeoCurriculum, a geometry-aware curriculum that orders learning by point-cloud-derived geometric similarity of tasks. Introduces GiGSL, an iterative generalist-specialist scheme that uses geometry to define task subspaces for specialists and distills them into a stronger generalist. Demonstrates the approach at scale (thousands of object instances) in a simulated table-top setting, recognized as an ICCV 2023 Best Paper finalist.

It learns a single policy that grasps thousands of diverse, randomly posed objects from point-cloud observations in a table-top scene. The project page shows training in IsaacGym over 3000+ object instances, success-rate training curves, geometry-based task-cluster visualizations, and videos of the robot hand grasping varied objects in different poses.
